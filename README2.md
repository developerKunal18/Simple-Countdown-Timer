# Simple Countdown Timer

A Python script that counts down from a given number of seconds.

## Description
This project takes a number of seconds as input and prints the countdown until it reaches zero.

## Features
- Countdown logic
- Uses time delay
- Beginner-friendly implementation

## Technologies Used
- Python
- time module

## How to Run
1. Clone the repository
2. Navigate to the project folder
3. Run:

   python countdown_timer.py

4. Enter countdown time in seconds

## Example

Input:
5

Output:
5
4
3
2
1
Time's up!
